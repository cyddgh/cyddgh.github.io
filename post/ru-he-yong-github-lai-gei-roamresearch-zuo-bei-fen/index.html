如何用Github来给RoamResearch做备份

2020-06-26 1 min read

我们要用到备份软件：https://github.com/MatthieuBizien/roam-to-git
在这里面就有详细的描述。但对于我一个还没用过GitHub action的外行人，中间还是遇到了不少麻烦。
具体步骤如下：
1. 建一个私人的GitHub库（也可以是公开的，弊端就是谁都能看到你的笔记），名称不一定是英文说明中的notes，
2. 需要配置你的Github secrets。你进入你所设立的库，在页面上方第三行（以code开头的那行）找到 settings（最后一个）点击，之后在左手边的列表找到的secrets（倒数第二个）。
3. 点击New secrets进行配置。需要设立3个值“ ROAMRESEARCH_USER”、“ ROAMRESEARCH_PASSWORD”和“ ROAMRESEARCH_DATABASE”分别对应你RoamResearch中用户名、密码和库的名称。
4. 之后参照演示站（ https://github.com/MatthieuBizien/roam-to-git-demo）建立一个main.yml，放在.github/workflows/，文字内容和演示站一致
5. 点击Github Action查看运行情况
